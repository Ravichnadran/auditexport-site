Documentation

Installation, usage, verification, and audit-safe operation of AuditExport.

Overview

AuditExport generates immutable, audit-ready technical evidence for ISO/IEC 27001 audits. It runs locally or in CI environments.

Installation

unzip auditexport-v0.1.1-macos.zip
chmod +x auditexport
mv auditexport /usr/local/bin/

Verify Release Integrity

sha256sum auditexport

Basic Usage

auditexport \
  --audit ISO27001 \
  --github \
  --output ./evidence

CI/CD Example

- name: Generate audit evidence
  run: |
    auditexport --audit ISO27001 --github --output evidence
