AuditExport

Generate ISO/IEC 27001 GitHub audit evidence automatically.

A minimal CLI tool for engineering and compliance teams to produce audit-ready technical evidence — fast, local, and deterministic.

What it does

  • Collects GitHub repository metadata using read-only access
  • Exports immutable JSON evidence files
  • Generates summaries, metadata, and hashes
  • Packages evidence into an audit-ready ZIP
  • Runs locally — no SaaS, no data upload

How it works

  • Download the CLI for your operating system
  • Create a read-only GitHub access token
  • Run a single command
  • Receive an audit-ready evidence ZIP

Supported in v0.1.1

  • ISO/IEC 27001
  • GitHub evidence collection
  • Linux (x86_64) and macOS
  • Local execution only

Roadmap

  • GitLab and Bitbucket collectors
  • AWS evidence collectors
  • Additional audit standards
